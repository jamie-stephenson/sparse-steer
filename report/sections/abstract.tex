\begin{abstract}
Activation steering modifies the internal representations of large language models at inference time to control their behaviour, but existing methods either require manual tuning of where and how strongly to intervene, or learn full intervention vectors at significant parameter cost. This project proposes \emph{sparse activation steering}, a hybrid that fixes steering directions from data and then learns only \emph{which} model components to steer and \emph{how much}, using a small number of trainable scalar parameters. We will evaluate the method on multiple behavioural benchmarks, comparing against previous targeted steering techniques and more general parameter-efficient fine-tuning baselines.
\end{abstract}
